Advanced Analytics Project

Introduction

This project introduced more advanced concepts, including linear regression, machine learning regression, decision trees, and clustering.

There was also exposure to R, including using ggplot for visualisation.

Other themes included machine learning and sentiment analysis.

About the Project

This project focused on a fictional games company that wanted to use predictive analytics to better understand its customer base and how they interacted with their loyalty program.

This culminated in a technical report that can be downloaded and read below.

Download PDF Report

Modules

1. Make predictions with regression

Apply linear regression to understand and predict how customers accumulate loyalty points.

2. Exploring data with decision trees

Use decision trees to reveal key splits and decision points that drive loyalty point accumulation.

3. Making predictions with clustering

Run k-means clustering to assess how remuneration and spending scores segment customers for analysis.

4. Analysing customer sentiments with reviews

Apply NLP to customer reviews to extract sentiment and insights that can inform marketing campaigns.

5. Cleaning, wrangling, and EDA in R

Load, clean, wrangle, and visualise data in R to build a robust analytical foundation.

6. Make recommendations to the business

Combine descriptive statistics and modelling results into actionable recommendations, visualised clearly.
