Market Analysis

This project explored a hypothetical supermarket with stores worldwide. The aim was to analyze sales patterns, customer demographics, marketing effectiveness, and conversion rates across various channels.

The data was cleaned and wrangled in Excel, and then imported into Tableau for visualisation. SQL was used to create additional datasets for analysis, and for a deeper dive into the data.

This project involved creating SQL databases, importing the tables from CSV files, and then using SQL to query the data, with exposure to grouping, aggregating, and joining tables.

A final PDF report was created to present the findings to the client, and includes a technical appendix.
