Market Analysis

Introduction

This project kicked off my data analyst journey. I worked with Excel, Tableau, and SQL to build strong foundations in data collection and cleaning.

I already use SQL day to day, but this module deepened my Excel skills—especially pivot tables—and how to turn them into effective dashboards.

Project Information

This was a hypothetical supermarket case with a global store footprint. The goal was to analyse sales patterns, customer demographics, marketing effectiveness, and conversion rates across channels.

Approach

Data was cleaned and wrangled in Excel, then imported into Tableau for visualisation. SQL was used to build additional datasets and support deeper analysis.

I created SQL databases, imported CSVs, and wrote queries using grouping, aggregation, and joins.

The findings were compiled into a client-ready PDF report with a technical appendix.
